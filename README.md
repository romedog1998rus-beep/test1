# Forge Signal Mapper

Новое Android-приложение для отметки радиосигнала на карте в тёмном техно-готическом sci-fi стиле. Это самостоятельный проект, не использующий официальные названия, логотипы или материалы Warhammer.

## Как собрать APK без Android Studio

1. Создай новый репозиторий на GitHub: https://github.com/new
2. Нажми **uploading an existing file**.
3. Перетащи содержимое этой папки в репозиторий.
4. Нажми **Commit changes**.
5. Открой вкладку **Actions**.
6. Выбери **Build Android APK**.
7. Нажми **Run workflow**.
8. Когда сборка закончится, открой последний запуск и скачай artifact **ForgeSignalMapper-debug-apk**.
9. Внутри будет `app-debug.apk` — это установщик Android.

## Разрешения

Приложение запрашивает доступ к геолокации и интернету для карты.
