КАК ПОЛУЧИТЬ APK БЕЗ ANDROID STUDIO:
1. Создай новый репозиторий на GitHub.
2. Зайди в репозиторий и нажми Add file -> Upload files.
3. Загрузи ВСЁ содержимое этой папки, не сам ZIP.
4. Нажми Commit changes.
5. Открой Actions -> Build Android APK -> Run workflow.
6. После завершения скачай artifact ForgeSignalMapper-debug-apk.
7. Внутри будет app-debug.apk — установщик Android-приложения.

Если GitHub попросит включить Actions, нажми Enable Actions.
